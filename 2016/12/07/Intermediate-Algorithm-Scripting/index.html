Intermediate Algorithm Scripting(FreeCodeCamp)

Posted on 2016-12-07 |

Intermediate Algorithm

1. Sum All Numbers in a Range

We’ll pass you an array of two numbers. Return the sum of those two numbers and all numbers between them.
The lowest number will not always come first.

2. Diff Two Arrays

Compare two arrays and return a new array with any items only found in one of the two given arrays, but not both. In other words, return the symmetric difference of the two arrays.

比较两个数组，返回两个数组的差异。

3. Roman Numeral Converter

Convert the given number into a roman numeral.

将给定的数字转换成罗马数字。

4. Wherefore art thou

Make a function that looks through an array of objects (first argument) and returns an array of all objects that have matching property and value pairs (second argument). Each property and value pair of the source object has to be present in the object from the collection if it is to be included in the returned array.

遍历一个对象数组（第一个参数）并返回一个包含相匹配的键-值对（第二个参数）的所有对象的数组。

5. Search and Replace

Perform a search and replace on the sentence using the arguments provided and return the new sentence.

使用给定的参数对句子执行一次查找和替换，然后返回新句子。

6. Pig Latin

Translate the provided string to pig latin.

把指定的字符串翻译成 pig latin。

7. DNA Pairing

The DNA strand is missing the pairing element. Take each character, get its pair, and return the results as a 2d array.

DNA 链缺少配对的碱基。依据每一个碱基，为其找到配对的碱基，然后将结果作为第二个数组返回。

Pairs: A-T, C-G, EX: CGC -> [[“G”, “C”], [“C”,”G”],[“G”, “C”]]

8. Missing Letters

  1. Find the missing letter in the passed letter range and return it.

  2. If all letters are present in the range, return undefined.

  3. 从传递进来的字母序列中找到缺失的字母并返回它。

  4. 如果所有字母都在序列中，返回 undefined。

9. Boo Who

Check if a value is classified as a boolean primitive. Return true or false.

检查一个值是否是基本布尔类型，并返回 true 或 false。

10. Sorted Union

Write a function that takes two or more arrays and returns a new array of unique values in the order of the original provided arrays.

写一个 function，传入两个或两个以上的数组，返回一个以给定的原始数组排序的不包含重复值的新数组。

11. Convert HTML Entities

Convert the characters &, <, >, “ (double quote), and ‘ (apostrophe), in a string to their corresponding HTML entities.

将字符串中的字符 &、<、>、” （双引号）, 以及 ‘ （单引号）转换为它们对应的 HTML 实体。

12. Spinal Tap Case

Convert a string to spinal case. Spinal case is all-lowercase-words-joined-by-dashes.

将字符串转换为 spinal case。Spinal case 是 all-lowercase-words-joined-by-dashes 这种形式的，也就是以连字符连接所有小写单词。

13. Sum All Odd Fibonacci Numbers

Given a positive integer num, return the sum of all odd Fibonacci numbers that are less than or equal to num.

给一个正整数num，返回小于或等于num的斐波纳契奇数之和。
